MyTravelApp

A mobile Travel Companion App designed to help international travellers quickly convert essential values such as currency, fuel efficiency and temperature.
This project focuses on simplicity, usability and real-world travel needs.


**Features**
Currency Converter:
Convert between multiple currencies using predefined exchange rates

Fuel Efficiency Converter:
Convert between km/L, MPG etc

Liquid Volume Converter:
Convert between liters, gallons

User-Friendly Interface:
Clean and minimal UI designed for quick access during travel

Offline Functionality:
Works without internet using fixed conversion values
